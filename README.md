# Archive Of Our Own Connector

Once feature complete, this jar will be used to build an android application that allows searching and reading of stories on Archive Of Our Own.
It provides an easily integrable API for anyone to create an app for AO3.

#### It is not yet production ready, contributions are welcome.

## Planned Features:
* API to get chapter-wise textual content.
* Implementing robust searching, filtering and sorting of entries based on user preferences.
* Implementing author-specific entry search and filtering.

## Built using:
* Kotlin and Gradle
* OkHttp and JSoup libraries for getting and parsing the AO3 pages.
